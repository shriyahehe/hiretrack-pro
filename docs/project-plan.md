# HireTrack Pro

## Overview

HireTrack Pro is a modern Applicant Tracking System (ATS) designed to help recruiters and hiring teams manage candidates, schedule interviews, and monitor the hiring pipeline from a single platform.

The application is built as a full-stack SaaS product using Next.js, TypeScript, Prisma, Supabase, and Tailwind CSS.

---

# Problem Statement

Recruiters often manage hiring using spreadsheets, emails, and multiple disconnected tools.

HireTrack Pro centralizes the recruitment process by providing:

- Candidate management
- Interview scheduling
- Hiring analytics
- Resume management
- Recruiter dashboard
